# 睡美人戰情室 - 客戶與業務管理規則

## 業務分組合併規則

### 當前生效的合併
- **陳育瑋** → 併入 **陳勁多**

## 客戶分配原則
- 當客戶從業務 A 重新分配給業務 B 時，該客戶的**所有銷貨記錄**（不限時間範圍）都要計算在業務 B 名下
- 後端需維護「客戶→業務」的對應表，優先以最新的分配為準

## 客戶名稱歸一化規則

### 前字匹配（前二字相同即為同一家）
- **專岩** → 「專岩實業」與「專岩空間」視為同一客戶，統一掛「專岩」
- **禾昇** → 「禾昇總合」與「禾昇綜合」統一掛「禾昇」

### 需合併的客戶清單
| 合併前 | 合併前 | 統一掛號 |
|------|------|--------|
| 信義星 | 太爾 | **信義星** |
| 琮達 | 琮威 | **琮威** |

## 統計篩選規則

### 交易筆數下限
- **交易筆數 < 15 筆**：不列入統計（視為單一案件客戶，排除）

### 出貨 vs. 樣品
- **只計出貨**，不計樣品
- 樣品判定邏輯（自 `isSampleRow()`）：
  - SKU 代號結尾為 `-S` 或 `-S1`
  - 備註含「樣品」、「陳列」、「SAMPLE」、「扣帶」等關鍵字
  - 金額為 0 的列

## 業務績效分析

### 按業務分組篩選
- 網頁提供選擇框，可按業務篩選查看該業務全部客戶狀況
- 顯示項目：
  - 進成客戶（YOY ≥ 10%）
  - 退步客戶（YOY ≤ -30%）
  - 90 天未下單
  - 半年未下單

### 排序選項
- **YOY（年同比成長率）** ↑ / ↓
- **最後下單日期** ↑ / ↓
- **今年業績** ↑ / ↓
- **到訪次數** ↑ / ↓
