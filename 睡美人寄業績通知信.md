## 程式架構

- `classes/traits/MailTrait.php` — 發信核心
  - `sendDailyPerformanceReport($recipients, $fromEmail, $type)` — `$type` 可為 `group`（三家合一）或 `main`（僅高雅瓷）
  - 依序讀三家的 `經銷銷售報表` Sheet（各公司有自己的 Google Sheet ID）
  - 統計當日銷貨、當月累計、客戶排行 Top 10、業務統計、系列 Top 5
  - 用 PHP 的 `mail()` 送出 HTML 郵件

- `api.php` — API 觸發點
  - `action=send-daily-email&type=group|main&token=...&to=...`
  - `token` 需等於 `CRON_TOKEN`（`sb-cron-2026-rebuild`）
  - `to` 可蓋掉 `config.php` 的預設收件人

- `config.php` — 設定
  - `DAILY_EMAIL_TO` — 預設收件人
  - `DAILY_EMAIL_FROM` — 寄件人顯示名稱
  - `SS_ID_MAIN` / `SS_ID_ANDYGA` / `SS_ID_XIYENA` — 三家公司試算表 ID

## cPanel Cron Job 設定

在 cPanel → Cron Jobs 設兩個排程：

```
30 19 * * 1-5 curl -s "https://bigt.cc/sleeping-beauty/api.php?action=send-daily-email&type=main&token=sb-cron-2026-rebuild&to=sb780910@gmail.com,ghosts0125@gmail.com,panus081231@gmail.com"
0 21 * * 1-5 curl -s "https://bigt.cc/sleeping-beauty/api.php?action=send-daily-email&type=group&token=sb-cron-2026-rebuild&to=titankou2002@gmail.com,lynnloveshumi@gmail.com"
```

- `30 19` = 晚上 7:30
- `0 21` = 晚上 9:00
- `1-5` = 週一至週五（假日不寄）
- 部署後若改過 `MailTrait.php`，需 push 上 GitHub 讓 Actions 自動 deploy 到伺服器
